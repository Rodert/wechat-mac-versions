## Как Скачать Старые Установщики

> Подробная инструкция доступна на [странице Wiki](https://github.com/Rodert/wechat-mac-versions/wiki/%E5%A6%82%E4%BD%95%E4%B8%8B%E8%BD%BD%E5%8E%86%E5%8F%B2%E5%AE%89%E8%A3%85%E5%8C%85-instructions).  
> Для просмотра releases онлайн откройте [GitHub Pages](https://rodert.github.io/wechat-mac-versions/), где перечислены все архивные версии с прямыми ссылками на загрузку.

Этот репозиторий хранит старые установочные пакеты WeChat через GitHub Releases. Чтобы скачать нужную версию:

1. Откройте [страницу Releases](https://github.com/Rodert/wechat-mac-versions/releases) этого репозитория.
2. Найдите нужную версию в списке releases. Список отсортирован в обратном хронологическом порядке, поэтому самая новая версия находится сверху.
3. Откройте страницу соответствующего Release.
4. Скачайте установочный файл `.dmg`.
5. При необходимости скачайте файл `.sha256`, чтобы проверить целостность установщика.

## Примечания

- Обновление от 2024-10-01: проект теперь конвертирует DMG в IMG и распаковывает его, чтобы получить точный номер версии WeChat, например `3.8.9.xx`. До этого обновления последние две цифры минорной версии получить было невозможно, поэтому добавлялся суффикс с датой обновления, чтобы различать минорные сборки внутри одной основной версии. Перед загрузкой старых версий проверяйте их внимательно.

Проект использует GitHub Actions для автоматической загрузки последнего официального установщика WeChat, расчета его hash-значения и публикации в GitHub Releases. В настоящее время архивировать установщики из App Store не планируется.

Журнал изменений версий доступен в официальном [changelog](https://weixin.qq.com/cgi-bin/readtemplate?lang=zh_CN&t=weixin_faq_list&head=true) WeChat.

Если у вас есть вопросы или претензии, связанные с нарушением прав, откройте issue.
